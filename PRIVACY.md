# Litmark Privacy Policy

Last updated: July 13, 2026

Litmark is a browser extension that lets you highlight text on web pages and
have those highlights reappear when you revisit them.

## What Litmark stores

- The text you highlight, a small amount of surrounding context (used to
  relocate the highlight if the page changes slightly), the page's URL and
  title, and the color you chose.

This data is stored using the browser's built-in `storage.local` API and
stays on your device. Litmark has no server, makes no network requests, and
does not use analytics, ads, or tracking of any kind.

## What Litmark does NOT do

- It does not transmit any data to Litmark's developer or any third party.
- It does not sell, share, or otherwise disclose your data to anyone.
- It does not collect browsing history beyond the pages you explicitly
  highlight text on.

## Why Litmark requests broad site access

Litmark needs to run on any page you choose to highlight, so its content
script requests access to all websites (`<all_urls>`). This access is used
only to read the page's visible text (to place and restore highlights) and
is never used to transmit page content anywhere.

## Data export / deletion

You can export your highlights to a local JSON file, or delete some or all
of them, at any time from the extension's popup. Uninstalling the extension
removes all locally stored data.
